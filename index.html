Experiencia

Desarrollé este proyecto para crear un ejecutable en Windows que permita a los usuarios descargar videos de YouTube con la calidad deseada. Utilicé las librerías youtube-dl-exec y fluent-ffmpeg sobre Electron.js como práctica autodidacta en la manipulación de medios en línea.

Creado para gestionar la subida y manipulación de archivos, especialmente imágenes y videos. Utiliza Express.js como framework principal y ofrece rutas para operaciones de carga y recuperación de archivos, así como tareas programadas de limpieza para mantener el sistema ordenado y eficiente.

Desarrollé esta herramienta web para realizar solicitudes HTTP utilizando diferentes métodos. Permite a los usuarios enviar peticiones a una URL específica con opciones avanzadas como el uso de tokens de autorización, envío de datos y carga de archivos.

En desarrollo, Lector es una aplicación diseñada para simplificar la conversión de texto en voz de manera personalizable. Utilizando la tecnología Electron, se pretende que Lector sea una aplicación de escritorio multiplataforma que mejore la accesibilidad y la experiencia de usuario en la lectura de textos largos y complejos.
